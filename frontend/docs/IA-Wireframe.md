# NovaTerra - IA & Wireframe 문서

## 1. Information Architecture (IA)

### 사이트 구조

```
NovaTerra
│
├── 홈 (/)
│   ├── Hero Section
│   ├── 플랫폼 통계
│   ├── 핵심 기능 소개
│   ├── 투자 가능 부동산 미리보기
│   ├── 핵심 기능 리스트
│   ├── CTA Section
│   └── Footer
│
├── 부동산 (/properties)
│   ├── 카테고리 필터
│   ├── 부동산 그리드
│   └── 부동산 상세 (/properties/:id)
│       ├── 부동산 이미지
│       ├── 프로젝트 소개
│       ├── 상세 정보
│       └── 투자 패널
│
├── 포트폴리오 (/portfolio)
│   ├── 투자 통계 대시보드
│   ├── 보유 부동산 목록
│   └── 거래 내역
│
├── 마켓플레이스 (/marketplace)
│   ├── 거래 통계
│   ├── 판매 중인 토큰 리스트
│   └── 판매 등록 CTA
│
└── 거버넌스 (/governance)
    ├── 투표 통계
    ├── 제안 목록
    └── 제안 올리기 CTA
```

## 3. 네비게이션 플로우

```
[홈페이지]
   ├─→ [투자 시작하기] → (회원가입/로그인)
   ├─→ [부동산 카드 클릭] → [부동산 상세]
   ├─→ [전체 보기] → [부동산 목록]
   └─→ [Navigation 메뉴]
        ├─→ [부동산] → [부동산 목록]
        │               └─→ [카드 클릭] → [부동산 상세]
        │                                  └─→ [토큰 구매]
        ├─→ [포트폴리오] → [거래 내역 확인]
        ├─→ [마켓플레이스] → [토큰 구매] or [판매 등록]
        └─→ [거버넌스] → [투표하기] or [제안 올리기]
```

## 4. 주요 사용자 플로우

### 4.1 신규 투자자 플로우
```
홈페이지 방문
  ↓
플랫폼 정보 확인 (통계, 기능)
  ↓
부동산 목록 탐색
  ↓
부동산 상세 확인
  ↓
회원가입/KYC 인증
  ↓
토큰 구매
  ↓
포트폴리오 확인
```

### 4.2 기존 투자자 플로우
```
로그인
  ↓
포트폴리오 확인 (수익 현황)
  ↓
[선택지 1] 추가 투자 → 부동산 목록
[선택지 2] 토큰 판매 → 마켓플레이스
[선택지 3] 의사결정 → 거버넌스 투표
```

### 4.3 거버넌스 참여 플로우
```
거버넌스 페이지 접근
  ↓
제안 목록 확인
  ↓
제안 상세 읽기
  ↓
찬성/반대 투표
  ↓
투표 완료 확인
```

### 5.1 카드 타입

│   [이미지]    │ ← 부동산 이미지

#### Proposal Card (제안 카드)
```
┌──────────────────┐

│ 부동산 이름      │

│                  │
│ 투표 참여율      │
│ [프로그레스바]   │
│                  │
│ [찬성] [반대]    │
└──────────────────┘
```

### 5.2 버튼 타입

- **Primary Button**: 그라데이션 배경 (투자, 구매, 투표)
- **Secondary Button**: 아웃라인 (상세보기, 취소)
- **Ghost Button**: 텍스트 (로그인)
- **Filter Button**: 토글 형식 (카테고리 필터)

### 5.3 색상 시스템

- **Primary**: 보라-파랑 그라데이션 (CTA, 강조)
- **Success**: 초록 (수익률, 찬성)
- **Destructive**: 빨강 (반대, 경고)
- **Accent**: 노랑-주황 (가격, 중요 정보)
- **Muted**: 회색 (보조 정보)
